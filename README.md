

# Basic Next.js Project

This is a **very basic Next.js project** designed for learning purposes. It helps you understand the core features of Next.js and build simple applications.

## Features

* Learn **Next.js routing** and **nested routing**
* Create and use **API routes**
* Simple project structure for beginners

> ⚠️ **Note:** This project currently does **not** have an authentication system or private route system. These features will be added soon.

## Getting Started

### Prerequisites

* Node.js installed (v16 or above recommended)
* npm (comes with Node.js)

### Installation

1. **Clone the repository**

```bash
git clone <your-repo-url>
```

2. **Navigate to the project folder**

```bash
cd <project-folder-name>
```

3. **Install dependencies**

```bash
npm install
```

4. **Run the development server**

```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the project.
